TypeScript gyakorlás

Projekt generálás

Készíts egy TypeScript-es vite projektet. A projekt segítségével plüssállatok adatait lehet megadni.

Form

Készíts űrlapot, amivel az alábbi adatokat lehet megadni:

· A plüss neve

· Magassága (cm)

· Színe (előre megadottak közül)

Osztály

Készíts egy osztályt, amely az adatokat eltárolja.

Hibás adatokat kivétellel jelezd:

· Üres név

· 0 vagy negatív magasság

· Ha a szín nem a megadottak közül lett kiválasztva

Összekötés

A form elküldésekor hozz létre egy példányt az osztályból. Figyelj a típusokra!

A kivétel tartalmát jelenítsd meg a felhasználónak!

Retool API

Hozz létre egy teszt API-t. Letrehozás után küldd el az adatokat az API POST végpontjára!

Listázás

Az oldal betöltésekor ill. az új plüss létrehozásakor írd ki a létező plüssök adatait, táblázatosan a form alá.

Letöltéskor használd az „as” műveletet, hogy a listát „Plüss”-ökből álló listává alakítsd!
